# MySparql
sparqlを用いた動的のような静的webサイト試作

## 実装アプリ一覧

### 1. [アイドル・人物検索 (im@sparql使用)](https://hagiayato.github.io/MySparql/imasparql/idolsearch/)

- **概要**  
  アイドルマスターシリーズのアイドルや関連人物情報を、im@sparqlのRDFデータを用いて検索・表示できるWebアプリです。
- **主な機能**  
  - アイドル・人物の検索
  - 詳細プロフィール表示（誕生日、血液型、キャストなど）
  - 衣装・ユニット情報表示
  - 関連リンク生成（Wikipedia, RDFリンク等）

---

### 2. [Stage for Cinderella グループ分けシミュレーション](https://hagiayato.github.io/MySparql/sfcinderella/)

- **概要**  
  シンデレラガールズのアイドルを属性や年齢などの条件でグループ分けし、シミュレーションできるWebアプリです。
- **主な機能**  
  - 条件指定によるグループ分けシミュレーション
  - グループごとのメンバー表示
  - 結果の再計算・リセット

---

## 備考

- 各アプリは静的HTML/JSで構成されており、GitHub Pagesで公開しています。
- 本プロジェクトは有志による非公式ファンツールです。
