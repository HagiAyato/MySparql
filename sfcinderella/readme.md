# Stage for Cinderella グループ分けシミュレーションWebアプリ

## 概要

このWebアプリは「Stage for Cinderella」企画向けに、アイドルマスター シンデレラガールズのアイドルを様々な条件でグループ分け・シミュレーションできるツールです。  
`sfcinderella` フォルダ配下の各HTML/JSファイルが画面・機能ごとに分かれており、グループ分けの条件設定や結果表示を行います。

---

## 主な機能

- **アイドルのグループ分けシミュレーション**
- **属性・年齢・身長などの条件指定**
- **グループごとのメンバー表示**
- **結果の再計算・リセット**

---

## フォルダ構成

```
sfcinderella/
├─ index.html           ... トップページ・シミュレーション画面
├─ result.html          ... グループ分け結果表示画面
├─ script/
│   ├─ common.js        ... 共通処理JS（UI操作・データ処理など）
│   ├─ simulation.js    ... シミュレーションロジック
│   ├─ result.js        ... 結果表示用JS
└─ style.css            ... 共通スタイル
```

---

## 使い方

1. `index.html` をブラウザで開き、グループ分け条件を設定します。
2. 「シミュレーション開始」ボタンを押すと、条件に応じたグループ分け結果が表示されます。
3. 結果画面では各グループのメンバー一覧を確認できます。
4. 条件を変更して再シミュレーションも可能です。

---

## 技術情報

- **フロントエンド**: HTML, CSS, JavaScript (jQuery)
- **共通処理**: `script/common.js` にてUI操作やデータ処理を関数化
- **シミュレーションロジック**: `script/simulation.js` にてグループ分けアルゴリズムを実装
