Stage for Cinderella グループ分けシミュレーション


説明

概要

アイドルマスターシンデレラガールズの投票イベント、Stage for Cinderella 予選のグループ分けシミュレーションページです。
イベント予想や、戦略を考えることにお役立てください。
想定条件：
  • 投票対象アイドル数：190人
  • グループ数：4グループ
  • グループ分け条件：特になし(完全ランダム) ※本サイト作者の予想

使い方

  1. グループ分けボタンを押すと、190人のアイドルを4グループに分けた時のグループ分けを表示します。
  2. グループ分けボタンを押す度に、結果が変わります。

表示の補足

  1. 属性：アイドルの属性
    (Cu=キュート/Co=クール/Pa=パッション)
  2. 名前：アイドルの名前
  3. ボイス：アイドルのボイス有無
  4. 圏内入り経験：過去の投票イベントでの圏内入り有無(中間、速報含む)
    出典： 第10回シンデレラガール総選挙 投票結果 ニコニコ大百科
  5. CG(1位)経験：CG=シンデレラガール(1位)経験有無

グループ一覧

グループA

属性 名前 ボイス 圏内入り経験 CG(1位)経験

グループB

属性 名前 ボイス 圏内入り経験 CG(1位)経験

グループC

属性 名前 ボイス 圏内入り経験 CG(1位)経験

グループD

属性 名前 ボイス 圏内入り経験 CG(1位)経験
